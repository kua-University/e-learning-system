# Project Overview

The E-Learning System is a full-stack web application developed to support online learning activities such as user registration, login, course creation, and enrollment. The system follows layered architecture and modern full-stack development principles.



# Technologies & Tools Used

- React.js
- Node.js
- Express.js
- SQLite
- Docker

## Development Environment

- Command Prompt / VS Code Terminal

- Component-Based Architecture

- MVC Pattern

# Features

- User Registration

- REST API Communication
- Responsive User Interface
- Docker Containerization

### How to run the project 

## Run Normally

# For Backend write these commands in the terminal
- cd backend
- npm install
- node server.js

# For Frontend write these commands in the terminal
- cd frontend
- npm install
- npm start

## Run with Docker

- docker-compose up --build

# Frontend
http://localhost:3000

# Backend
http://localhost:5000
