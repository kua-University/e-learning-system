# E-Learning System
📌 Overview
This is a web-based E-Learning System that allows users to register, log in, create courses, and enroll in available courses. The system is built using a modern full-stack approach with a clear separation between frontend, backend, and database.

## Technologies
- React.js (Frontend)
- Node.js + Express (Backend)
- SQLite (Database)

## How to Run:

### for the Backend write the following commands in the terminal
- cd backend
- npm install
- node server.js

### Frontend write the following commands in the terminal
- cd frontend
- npm install
- npm start

## Features
- User Registration & Login
- Course Creation
- Enrollment System

 ### Tools Used
- Visual Studio Code (for development)
